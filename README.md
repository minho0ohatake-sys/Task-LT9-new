## 🧮 Counter 

* Tăng / giảm giá trị counter
* Reset counter về 0
* Chọn bước nhảy (step): 1 / 5 / 10
* Giới hạn giá trị Min / Max
* Disable nút khi không thể tăng / giảm thêm

---

## ✅ Todo 

* Thêm công việc mới
* Validate không cho thêm todo rỗng
* Hiển thị danh sách todo bằng `map`
* Toggle hoàn thành công việc
* Xóa công việc
* Thống kê:

  * Tổng số công việc
  * Đã hoàn thành
  * Còn lại
* Filter:

  * All
  * Active
  * Completed
* Chỉnh sửa nội dung todo trực tiếp

  * Click vào title để edit
  * Enter để lưu
  * Blur để lưu
* Priority:

  * Low
  * Medium
  * High
* Hiển thị badge màu theo priority
* Highlight todo có độ ưu tiên cao

## 📦 Installation

Clone project:

```bash
git clone <repository-url>
```

Di chuyển vào thư mục project:

```bash
cd react-dashboard-app
```

Cài dependency:

```bash
npm install
```

Chạy môi trường development:

```bash
npm run dev
```

Build production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
